# PyWordle

Basic command-line Python implementation of the game Wordle.

# Features
 - Infinite gameplay (gives user ability to start a new game after one is finished)
 - Text-colouring to emulate Wordle functionality
 - Tutorial for those who do not know how to play Wordle

# Installation
1. Download source files.
2. Have Python installed on your computer (recommended is through Windows Store).
3. Run `python wordle.py` in the directory where the source files are stored.
4. Have fun!
